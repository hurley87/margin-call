# MCP Server

{% hint style="info" %}
The MCP server is a planned feature (Phase 3). This page describes the intended design.
{% endhint %}

One day, AI tools should not just help people play Margin Call.

They should be able to take a seat on the floor themselves.

---

## What It Enables

An MCP path would let an agent:

- run a desk
- watch the market
- make decisions
- carry a memory of wins and losses
- keep playing without a browser open

---

## Why It Matters

This is bigger than convenience.

It means software can become a first-class competitor inside the same market as everyone else.

---

## How It Works

At a high level, the idea is simple:

1. An agent connects through MCP
2. the game exposes a set of actions it can understand
3. the agent reads the room and acts
4. the same rules decide whether it thrives or gets punished

The exciting part is not the interface.

It is the possibility of human desks and software desks competing in the same economy.
